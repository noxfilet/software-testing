# Software-testing

[![Coverage Status](https://coveralls.io/repos/github/noxfilet/software-testing/badge.svg?branch=main)](https://coveralls.io/github/noxfilet/software-testing?branch=main)

E-commerce utility library testing project for COMP.SE.200.

## Selected libraries for testing (in `src/` )
1. add.js
2. capitalize.js
3. countBy.js
4. drop.js
5. eq.js
6. filter.js
7. get.js
8. isEmpty.js
9. keys.js
10. words.js

## Run the tests locally
### 1. Clone the repository
```bash
git clone https://github.com/noxfilet/software-testing/
cd software-testing
```
### 2. Install dependencies
```bash
npm install
```
### 3. Run tests
```bash
npm test
```
Coveralls report can be found at [Coveralls](https://coveralls.io/github/noxfilet/software-testing?branch=main)
